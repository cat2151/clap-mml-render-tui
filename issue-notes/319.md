# issue patch fileが見つからないとき、patches_factoryとpatches_3rdpartyを付与して検索する #319
[issues #319](https://github.com/cat2151/clap-mml-render-tui/issues/319)


- これまでの課題：
    - 音色をjsonで指定するとき、jsonのpatch nameの記述で、patches_factoryやpatches_3rdpartyというディレクトリ名を省略して記述したとき、patch fileが見つからず、UXが悪い
- 対策：
    - patch fileが見つからないとき、patches_factoryとpatches_3rdpartyを付与して検索する
    - そして検索して見つかった場合は、それをjsonにも反映する
        - つまり曖昧なjson表記を、検索結果で補正する
