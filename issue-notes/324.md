# issue オフラインレンダリングの並列実行statusを画面下部に表示する #324
[issues #324](https://github.com/cat2151/clap-mml-render-tui/issues/324)

- これまでの課題：直列にキューに入れてオフラインレンダリングしているように見受けられる
    - 処理の内容的には、並列で高速化できるのではないか？
    - 高速化すれば待ち時間が減ってUXが改善できる
- 対策：並列オフラインレンダリングを実装する、検証用に実装する
- 対策：並列の可視化を行う。並列status行を画面下部に1行追加で用意する
    - 現在いくつの並列数でrender中か、renderが終わって並列0か、直列なので並列数が1か、を可視化する
    - これはnotepadモードとdawモードそれぞれで表示する
