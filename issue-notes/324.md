# issue オフラインレンダリングの並列実行statusを画面下部に表示する #324
[issues #324](https://github.com/cat2151/clap-mml-render-tui/issues/324)

- これまでの課題：notepadモードで今、どれくらいオフラインレンダリングをしている途中か？が見えない
- 対策：並列オフラインレンダリングstatusの可視化を行う。
    - 並列status行を画面下部に1行追加で用意する
    - 現在いくつの並列数でrender中か、renderが終わって並列0か、を可視化する
    - これはnotepadモードとdawモードそれぞれで表示する
