# issue ランダム音色選択が運用としてトラブルの元になっているので対策を検討する #97
[issues #97](https://github.com/cat2151/clap-mml-render-tui/issues/97)

- 問題
    - tomlでoffにしないと、音色選択menuが使えない
    - これのせいで、blueskyからの利用も、非DAWからの利用も、直感的でなく、わかりづらい
        - より正確には、非ランダム音色を選ぼう、という変更がしづらく、UXが悪い
- 対策案
    - TUIでランダム音色をon/off可能にする。それがonのときのみ、ランダム音色にする。
    - tomlからランダム音色設定を削除する。これが運用ミスの原因だった。
