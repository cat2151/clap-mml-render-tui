# issue Notepadモードに切り替えた瞬間にauto updateが走ってuserが混乱した #321
[issues #321](https://github.com/cat2151/clap-mml-render-tui/issues/321)

- これまでの課題：Notepadモードに切り替えた瞬間にauto updateが走ってuserが混乱した
- 対策：auto update（起動時にbackgroundでremoteのhashをチェックする、独自実装）を廃止し、すべてのself update処理はcat-self-update-libクレートに委譲する
    - 具体的には、updateサブコマンドと checkサブコマンドを、cat-self-updateリポジトリのcat-self-update-libクレートを利用して実装する。
    - また、args解析にはRustのスタンダードなライブラリクレートを利用すること。
- 最新状況では、いくつかがほかのissueで実装済みとなっている可能性がある
    - まず、上記がすべて実装されているか？をチェックすること。すべて実装されていればOK。
        - もし一部でも実装されていないなら、その未実装部分を実装すること。
        - もし一部の廃止ができていないなら、その未廃止部分を廃止すること。
