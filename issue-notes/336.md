# issue カーソル点滅について、fg文字色の輝度を時間で変更（つまりfg文字色が点滅）するのをやめる。そしてtui-textareaを使うすべての処理でカーソル表示を縦線（点滅）にする #336
[issues #336](https://github.com/cat2151/clap-mml-render-tui/issues/336)

- これまでの課題：
    - fg文字色の輝度が時間で変更（つまりfg文字色が点滅）しており、目立ちすぎて、UXが悪かった
    - tui-textareaでは、カーソルがエスケープシーケンスで「ブロック（点滅なし）」になっているようだが、これではまるでvimのノーマルモードのようであり、状態がわかりづらく、userが混乱していた
- 対策
    - fg文字色の輝度の時間による変更（つまりfg文字色の点滅）をやめる
    - 一方でtui-textareaを使うすべての処理において、tui-textareaのカーソル表示は、エスケープシーケンスで「縦線（点滅あり）」に修正する
