# issue 絞り込みキーワード入力中ならカーソル点滅は絞り込みキーワード入力欄だけにする #334
[issues #334](https://github.com/cat2151/clap-mml-render-tui/issues/334)

- これまでの課題：
    - 絞り込みキーワード入力中、左paneのlistでカーソルが点滅し、右paneのlistでカーソルが点滅し、そして絞り込みキーワード入力欄ではカーソル点滅していなかった
    - 必要なのは、絞り込みキーワード入力欄（tui-textarea）でのみカーソル点滅することである
- 対策：
    - 絞り込みキーワード入力中、
        - 左paneのlistも、右paneのlistも、
            - fg文字色点滅をしない
            - 現在行のbgを明るくしない
        - そして絞り込みキーワード入力欄のtui-textareaでのみ、現在カーソルがある位置について、カーソル点滅（エスケープシーケンス利用、カーソル縦線、点滅あり）を行う
