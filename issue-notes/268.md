# issue gキーでgenerate、を実装する #268
[issues #268](https://github.com/cat2151/clap-mml-render-tui/issues/268)

- 目的。起動後にワンキーで音が鳴る、という体験を提供する
- 方法（MML generate）：
    - notepadモード： 現在行とその上の行の間に、1行挿入する。ランダム音色。MML挿入（後述）。
    - DAWモード : 現在カーソルのあるtrackのmeasに、MMLを挿入（後述）し、init measにランダム音色を挿入する。
    - notepadモードも、DAWモードも、挿入直前にhistoryにMML退避する。
        - 挿入したらpreview演奏する。
        - 挿入するMMLは、デフォルトはc1または`cfg1`のランダムで、SHIFT+Gで設定可能（後述）。
- 方法（MML generator settings）：
    - notepadモードとDAWモードどちらのモードでも、
        - SHIFT+Gキーで generator settings overlay を開く。
            - generator settings overlayは、
                - jkキーで選び、iキーで、generateするMMLを、userが文字列入力してENTERかESCで確定、で設定可能にする（デフォルトは`c1`と`'cfg1'`）。
                - 行追加はoとO、vim風。行削除はdd、vim風。generator settings overlayを抜けるのはESCキー。ENTERはjキーと同様（vim風）。jkキーで選んだときはそれをpreview演奏する。
                - 「?」入力で、generator settings overlay helpをoverlayで表示し、keybindsを表示する。
- 方法（help）：
    - helpに、gキーとSHIFT+Gキーの説明を追加する
- 備忘：
    - おそらくagentがcontext windowあふれでハルシネーションして実装漏れするので、これはissue-notesに書いておき、あとで参考にして漏れた部分を実装させていく。
