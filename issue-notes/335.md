# issue historyやtone selectの3種類のoverlayにおいて、現在focusのあたっているpaneだけ現在行のbgを明るくして目立たせるようにする #335
[issues #335](https://github.com/cat2151/clap-mml-render-tui/issues/335)

- これまでの課題：
    - 左右のpaneがどちらも、現在行のfg文字色が点滅しており、userが混乱した
    - 左右のpaneがどちらも、現在行のbgが明るくなっており、userが混乱した
- 対策：
    - 文字色点滅をやめる
    - 左右のpaneのうちfocusがあたっているほうだけ、
        - 現在行のbgを明るくして目立たせるようにする
        - （bgを明るくして目立たせる実装は現状どおりのはず）
    - カーソルがあたっていない側のpaneは、現在行のbgを明るくしないようにする
