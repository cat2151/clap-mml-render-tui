# issue self updateは、cat-self-update-libクレートの機能のみを利用するよう修正する #328
[issues #328](https://github.com/cat2151/clap-mml-render-tui/issues/328)

- これまでの課題：
    - `updater.rs` をみたところ、自前のバグっているself-update機能が多数残っており、それが使われてるように見受けられる
    - これまでの指示についてagentが理解できておらず、ハルシネーションにより誤った実装をしている
- 対策：
    - `updater.rs`と、それを呼び出すシステム、つまりself updateと関連機能を、全面的に修正する
    - Rustのclapクレートを利用してargs解析を全面的に修正する
    - 実装方法はcat-self-updateリポジトリを参考にすること
    - これまでのムダなself-update機能（バグっている）は全面的に削除すること
