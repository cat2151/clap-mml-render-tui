# issue DAWモード、patch historyでjkで音色を選んだらその音色でpreview演奏する #320
[issues #320](https://github.com/cat2151/clap-mml-render-tui/issues/320)

- これまでの課題：
    - DAWモードのpatch historyほか3つのoverlayにおいて、jkで音色を選んでも、cacheされたwavを演奏しているだけであり、意味がなかった
- 対策：
    - DAWモードは、patch historyほか3つのoverlayでjkで音色を選んだら、その音色を使って（現在のtrackのinit meas.の音色ではなく）preview演奏する
