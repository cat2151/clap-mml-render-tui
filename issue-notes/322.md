# issue history系overlayにては、spaceでpreview演奏できるようにする #322
[issues #322](https://github.com/cat2151/clap-mml-render-tui/issues/322)

- これまでの課題：
    - NotepadモードとDAWモードそれぞれの3つずつのhistory系overlay（tone select、patch phrase、notepad history）にて、
    - spaceを押しても何もfeedbackがなく、userが混乱した、ということがあった。
    - overlayによってspaceによるpreview演奏があったりなかったりで、
    - 不統一でuserが混乱した。
- 対策：
    - history系overlayにて、spaceでpreview演奏できるようにする
    - あらゆる場所において「spaceが、フィルタ入力等で既にkeybindsがあるなら、フィルタ入力等の既存keybindsを優先する」「それ以外の場合はspaceでpreview演奏する」
