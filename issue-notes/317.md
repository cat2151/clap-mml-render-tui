# issue patch select画面で、sort順序をCTRL+Sで変更できるようにする #317
[issues #317](https://github.com/cat2151/clap-mml-render-tui/issues/317)

- 前提：
    - patches_factoryと、patches_3rdparty の2つのディレクトリにpatchがある
    - patches_factoryは、patches_factory/pad のように、直下にカテゴリがある
    - patches_3rdpartyは、patches_3rdparty/john/pad のように、3rdparty_nameのあとにカテゴリがある
- sort順序その1：カテゴリ名sort
    - 以下の順番で並ぶ：
        - patches_factory/lead/super_lead
        - patches_3rdparty/john/lead/great_lead
        - patches_factory/pad/super_pad
        - patches_3rdparty/john/pad/great_pad
    - つまり、/lead/ /pad/ というカテゴリでsortされたのち、factoryの patch_name 、3rdpartyの  3rdparty_name + patch_name の順でsortされる
- sort順序その2：単純sort
    - 以下の順番で並ぶ：
        - patches_factory/lead/super_lead
        - patches_factory/pad/super_pad
        - patches_3rdparty/john/lead/great_lead
        - patches_3rdparty/john/pad/great_pad
    - つまり、単純にpatch path全体の文字列sortで並ぶ
